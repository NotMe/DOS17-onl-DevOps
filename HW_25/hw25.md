## 3

Создать свой первый pipeline по инструкции "Through the classic UI"

```
pipeline {
    agent any 
    stages {
        stage('Stage 1') {
            steps {
                echo 'Hello world!' 
            }
        }
        stage('Stage 2') {
            steps {
                sh 'hostname'
                sh 'whoami'
            }
        }
    }
}
```
